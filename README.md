# You-BilibiliFlv

![](https://img.shields.io/badge/npm-v1.1.0-brightgreen)

基于 flvjs 制作的 TSX 组件，支持选择性替换交互控件和蒙版画布的渲染逻辑。



## Install

```powershell
npm install you-bilibiliflv
```
